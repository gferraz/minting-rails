# Minting::Rails

Add Money attributes to your Rails application.


## Usage
Currently in development. Please wait for 1.0 release.

## Installation
Add this line to your application's Gemfile:

```ruby
gem "minting-rails"
```

And then execute:
```bash
$ bundle
```

Or install it yourself as:
```bash
$ gem install minting-rails
```
